Title:

I18N mon amour et Annotation Processing par l'exemple

Summary:

Nous allons vous raconter l'histoire d'une reprise en main de la localisation Java depuis le code source grâce à l'utilisation d'annotations et leur ami Mr Processeur à chaque chapitre.

Chapitre 1: Maman j'ai raté l'avion i18n
* L'histoire classique de la localisation a postériori d'une application. Mais avec l'introduction d'une annotation pour gérer les clefs et la langue de référence dans le code.

Chapitre 2: Mais où est I18n-Charlie ?
* Comment s'assurer que tout est localisé et que toutes les clefs sont utiles, générer un fichier excel pour revue.

Chapitre 3: GWT mon amour
* Comment partager les ressources texte simultanément dans GWT et J2EE

Chapitre 4: Mobile Me
* Coexistence d'une version web classique et web mobile (libellés simplifiés sur un petit écran), avec le 'deferred binding' de GWT.

Chapitre Bonus: i18n for ever
* La localisation de masse  (crowd sourcing) avec par exemple crowdin.net

Cette session se veut technique et ludique côté code et démonstrations.
